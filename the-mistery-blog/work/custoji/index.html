Emojis for everyone

In March 2020, during the height of lockdown in Ireland, I found myself climbing the walls ever so slightly. As a way to focus my energies I looked for side projects that would hopefully keep me distracted.

I find coding quite zen like. Writing logical statements, figuring out problems, tweaking shadows with CSS... I love it, and it helps me unwind to no end.

The result of this unwinding was Custoji. It’s a web app that lets you create any kind of emoji you like, using components built from Twitters twemojis pack.
